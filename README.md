# SMIT_BATCH_18_POST_APP
# 🚀 SMIT Batch 18 Post App

A modern social media post application designed for learning and hackathon practice. Users can create posts, view feeds, and interact with content in a clean and responsive interface.

---

## ✨ Features

- 📝 Create and publish posts
- 👤 User profile support with images
- ❤️ Like system for posts
- 📱 Responsive design for all devices
- 📰 Feed-style post browsing
- 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)

---

## ⚙️ Setup Instructions

1. Clone the repository:

git clone https://github.com/your-username/smit-post-app.git


2. Open project folder:

cd smit-post-app


3. Run project:
- Open `index.html` in browser
